- Freeze hidden shard generation and hash policy.
- Add fixed dependency lockfile or container for trusted replay.
- Define submission frequency limits and leaderboard probing policy.
- Add contamination and public-data leakage checks.
- Build trusted runner replay script.
- Define `limeslabs.eu` leaderboard ingestion for `templates/leaderboard-entry.json`.
- Publish baseline repeated-seed results.
- Add downstream mini-eval non-regression tasks.

## After First Trusted Replay

- Add replicated status for repeated-seed runs.
- Add scaling audit for larger token and model budgets.
- Publish negative and mixed result cards.
- Group leaderboard entries by method family, not only best score.
